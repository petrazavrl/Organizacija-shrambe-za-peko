# Organizacija shrambe za peko

To je moj projekt pri Uvodu v programiranje. 

1. Naredila bom bazo z nekaj recepti in možnostjo dodajanja receptov,
2. ko bom želela recept uporabiti se bo izpisal nakupovalni listek (v kolikor ne bo zadosti sestavin),
3. organizirala bom shrambo, da se bo ob nakupu prištelo kupljeno, ob peki pa odštelo porabljeno.
